# InVase (v0.1)

An Arduino based smart vase system with real time monitoring and automatic irrigation.

## Material list

 - Arduino IDE
 - Arduino Nano
 - Temperature and humidity sensor
 - Light sensor
 - Soil moisture sensor
 - Water level sensor
 - Vase with bottom
 - Water reservoir
 - Tube for water flow
 - Bluetooth module
 - Android smartphone
 - Mini water pump
 - Breadboard
 - Resistors
 - Diode
 - 9V battery adapter
 - Buck Converter (step-down)
 - Jumper wires
